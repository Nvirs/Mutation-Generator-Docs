# Mutation Generator Dokumentáció

Mutation Generator hivatalos dokumentációja

Itt talalja a projektet:

- tervezési es architekturális leirasát,
- API es feluleti dokumentaciojat,
- telepitési es futtatási útmutatóit,
- használati es tesztelési példáit.

## Gyors navigácio

- Tervezés es rendszerleíras: architektura és LLM részletek
- API es felületek: web API és log stream használat
- Környezet beállitasá: DVWA setup és VPS Docker demo
- Használat es tesztelés: gyors útmutato, manuális SQLi teszt, logok
